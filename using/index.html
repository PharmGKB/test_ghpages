Contributing

We welcome contributions from anyone willing to help improve PharmCAT.

If you notice a bug in PharmCAT, have an idea for a new feature, or just have a question about how PharmCAT works, please submit an issue in GitHub.

If you want to make a code contribution to the project please checkout the code repo and read the developer wiki for information about our development process.
